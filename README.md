# Python Programming Tasks

This repository contains a collection of Python programming exercises demonstrating fundamental programming concepts such as loops, functions, error handling, and algorithmic problem solving.

## Tasks Included

1. **Quiz Game**

   * Interactive quiz with five questions
   * Case-insensitive user input
   * Score calculation and feedback

2. **Number Swapping**

   * Demonstrates Python tuple unpacking
   * Swaps two user-entered numbers without a temporary variable
   * Handles invalid inputs gracefully

3. **Prime / Armstrong / Palindrome Identifier**

   * Checks whether a number satisfies any of these mathematical properties
   * Uses modular arithmetic, loops, and string reversal

4. **Fibonacci Sequence Generator**

   * Generates the first *n* Fibonacci numbers
   * Implemented iteratively using Python
